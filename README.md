# PNG to JPG Converter

A browser-based tool that converts PNG images to JPG format. Built with Next.js and processes files entirely in your browser.

ðŸ”— **Live Demo:** [PNG to JPG](https://toolsvana.com/tool/png-to-jpg)

## What it does

This tool converts PNG images to JPG format using the HTML Canvas API. Reduces file size significantly while maintaining good image quality. Note: Transparent areas become white.

## Key features

- Converts PNG to JPG format
- Reduces file size by 50-80%
- Maintains image quality
- Works offline
- No file uploads
- Instant conversion

## How to use

1. Drop a PNG file or click to browse
2. Conversion happens automatically
3. Download the JPG file

## Technical details

Built with Next.js 16, TypeScript, and Tailwind CSS. Uses HTML Canvas API for image processing with 92% quality setting.
